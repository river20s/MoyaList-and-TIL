# PEP 
PEP는 Python Enhancement Proposal의 약자로, 파이썬 커뮤니티에 정보를 제공하거나 파이썬이나 그 새로운 프로세스 또는 환경에 대한 새로운 기능을 설명하는 설계 문서이다.
